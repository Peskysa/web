Comandos en sistemas Linux

Guía básica para escribir un script en Bash

Paso 1: Crear un Archivo para el Script Bash con un editor de texto en tu sistema (como nano, vim o gedit en sistemas Unix/Linux) y guarda el archivo con una extensión .sh para indicar que es un script Bash. Por ejemplo, puedes llamarlo mi_script.sh. Paso 2: Escribe tu código Bash en el archivo .sh. Aquí tienes un ejemplo simple de un script Bash que saluda al usuario.


        #!/bin/bash

        # Este es un comentario
        echo "Hola, Mundo!"
  • #!/bin/bash: Esta línea, llamada shebang, indica que el archivo debe ser interpretado y ejecutado por el intérprete de Bash (/bin/bash en este caso).
  • # Este es un comentario: Los comentarios en Bash comienzan con # y son ignorados por el intérprete. Los comentarios son útiles para explicar el código y hacerlo más comprensible.
  • echo "Hola, Mundo!": Esta línea imprime "Hola, Mundo!" en la consola.

Antes de ejecutar el script, necesitas darle permisos de ejecución. Puedes hacerlo con el siguiente comando en la terminal. Esto otorga permisos de ejecución al archivo mi_script.sh.


        chmod +x mi_script.sh

Ahora que el script tiene permisos de ejecución, puedes ejecutarlo usando el siguiente comando en la terminal. Esto ejecutará el script y deberías ver la salida "Hola, Mundo!" en la terminal.


        ./mi_script.sh

Guía básica para usar comandos en Ubuntu y otras distribuciones basadas en Debian

Los sistemas basados en Unix/Linux como Linux y macOS utilizan una interfaz de línea de comandos (CLI) para interactuar con el sistema operativo. La mayoría de los comandos se ingresan en una terminal. A continuación, te mostraré cómo usar algunos comandos esenciales:

Para cambiar el nombre del equipo (hostname) en Linux, sigue estos pasos. Edita el archivo /etc/hosts para reflejar el cambio. Abre el archivo usando un editor de texto y cambia el nombre anterior.


        sudo nano /etc/hosts

Ahora ejecuta el siguiente comando, tienes que poner el mismo nuevo nombre en los dos sitios y ya estaría cambiado. Si cambiamos de consola con CONTROL+ALT y alguna de las siguientes: F1 F2 F3 F4 F5 F6 podemos comprobar que se ha cambiado el nombre del equipo sin reiniciar. Esto es importante porque en muchos server es necesario prestar servicio 24-7 todo el año. 


        hostnamectl set-hostname NOMBRE_NUEVO
